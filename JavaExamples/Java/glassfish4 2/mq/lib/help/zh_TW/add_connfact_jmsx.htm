JMSX 屬性

使用此標籤指定應受 Message Queue 設定的 JMS 定義的屬性。

儘管須耗用大量資源，但 Message Queue 仍支援許多 JMS 定義的屬性。若要節省資源，您應確定哪些屬性是用戶端應用程式真正需要的，並按一下相應的核取方塊啟用這些屬性的設定。

啟用 JMSXAppID 訊息屬性 指定 Message Queue 是否應對已產生的訊息設定 JMS 定義的屬性 JMSXAppID (發送訊息的應用程式的識別)。預設值：false

啟用 JMSXUserID 訊息屬性 指定 Message Queue 是否應對已產生的訊息設定 JMS 定義的屬性 JMSXUserID (發送訊息的使用者的識別)。預設值：false

啟用 JMSXProducerTXID 訊息屬性 指定 Message Queue 是否應對已產生的訊息設定 JMS 定義的屬性 JMSXProducerTXID (產生了此訊息的事務處理識別碼)。預設值：false

啟用 JMSXConsumerTXID 訊息屬性 指定 Message Queue 是否應對已產生的訊息設定 JMS 定義的屬性 JMSXConsumerTXID (用到此訊息的事務處理識別碼)。預設值：false

啟用 JMSXRcvTimestamp 訊息屬性 指定 Message Queue 是否應對已產生的訊息設定 JMS 定義的屬性 JMSXRcvTimestamp (訊息發送至使用者的時間)。預設值：false
