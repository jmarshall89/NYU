Manipulação de conexões 3.0

Utilize a aba Manipulação de conexões para reconfigurar informações básicas das conexões que serão criadas por esta fábrica.

Tipo de conexão Utilize a lista suspensa para selecionar o tipo de conexão.

Nome de host do broker Especifica o nome da máquina que hospeda o broker ao qual o cliente se conectará.

Porta de host do broker Especifica o número da porta do port-mapper. O cliente utiliza as informações fornecidas pelo port-mapper para se conectar ao serviço desejado, jms ou ssljms.

Porta de serviço do broker Especifica o número da porta que será utilizado para realizar conexão direta com o serviço jms ou ssljms. Utilize a Porta de serviço do broker para ignorar o port-mapper. Isso permite que os clientes se conectem por firewalls utilizando uma única porta conhecida.

URL HTTP Especifica o URL que será utilizado para se conectar ao Message Service. Por padrão, o URL é http://localhost/imq/tunnel.
