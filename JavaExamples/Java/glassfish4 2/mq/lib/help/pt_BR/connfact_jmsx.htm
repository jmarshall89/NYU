Propriedades JMSX

Utilize esta aba para especificar novamente que propriedades definidas pelo JMS devem ser configuradas pelo Message Queue.

O Message Queue oferece suporte a vários atributos definidos pelo JMS, porém em detrimento dos recursos. Para manter os recursos, você deve determinar quais destas propriedades são realmente necessárias para o aplicativo cliente e ativar a configuração de tais propriedades clicando na caixa de seleção apropriada.

Ativar a propriedade da mensagem JMSXAppID Especifica se o Message Queue deve configurar a propriedade definida pelo JMS, JMSXAppID (identidade do aplicativo que envia a mensagem), nas mensagens produzidas. Padrão: falso

Ativar a propriedade da mensagem JMSXUserID Especifica se o Message Queue deve configurar a propriedade definida pelo JMS, JMSXUserID (identidade do usuário que envia a mensagem), nas mensagens produzidas. Padrão: falso

Ativar a propriedade da mensagem JMSXProducerTXID Especifica se o Message Queue deve configurar a propriedade definida pelo JMS, JMSXProducerTXID (identificador da transação dentro da qual esta mensagem foi produzida), nas mensagens produzidas. Padrão: falso

Ativar a propriedade da mensagem JMSXConsumerTXID Especifica se o Message Queue deve configurar a propriedade definida pelo JMS, JMSXConsumerTXID (identificador da transação dentro da qual esta mensagem foi consumida), nas mensagens consumidas. Padrão: falso

Ativar a propriedade da mensagem JMSXRcvTimestamp Especifica se o Message Queue deve configurar a propriedade definida pelo JMS, JMSXRcvTimestamp (a data e hora em que a mensagem foi entregue ao consumidor), nas mensagens consumidas. Padrão: falso
