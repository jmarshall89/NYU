JMSX 属性

使用此选项卡，可以指定由 Message Queue 设置的 JMS 定义的属性。

Message Queue 支持多种 JMS 定义的属性，但却占用很多资源。要节约资源，应确定客户机应用程序实际需要其中的哪些属性，并通过单击相应的复选框启用那些属性的设置。

启用 JMSXAppID 消息属性 指定 Message Queue 是否应在产生的消息上设置 JMS 定义的属性 JMSXAppID（发送消息的应用程序的标识）。默认值：false

启用 JMSXUserID 消息属性 指定 Message Queue 是否应在产生的消息上设置 JMS 定义的属性 JMSXUserID（发送消息的用户标识）。默认值：false

启用 JMSXProducerTXID 消息属性 指定 Message Queue 是否应在产生的消息上设置 JMS 定义的属性 JMSXProducerTXID（产生此消息的事务的事务标识符）。默认值：false

启用 JMSXConsumerTXID 消息属性 指定 Message Queue 是否应在使用的消息上设置 JMS 定义的属性 JMSXConsumerTXID（使用此消息的事务的事务标识符）。默认值：false

启用 JMSXRcvTimestamp 消息属性 指定 Message Queue 是否应在使用的消息上设置 JMS 定义的属性 JMSXRcvTimestamp（消息传送到用户的时间）。默认值：false
