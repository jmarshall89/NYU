QueueBrowsers 和 ServerSessions

使用此选项卡，可以为客户机运行时重新配置队列浏览以及重新指定会话在应用程序服务器中的运行方式。

队列浏览器检索大小 指定客户机运行时在浏览队列目的地的内容时一次检索的最大消息数。默认值： 1000

队列浏览器检索超时（毫秒） 指定客户机运行时抛出异常之前在浏览队列目的地内容时等待检索消息的最大时间值。默认值：60000 毫秒。

加载最大的消息数到 ServerSessions 指定 Message Queue ConnectionConsumer 应该加载最大的消息数 maxMessages 到 ServerSession 的会话。不选中此复选框则指定 Message Queue ConnectionConsumer 一次只能装入一个消息。详细信息，请参阅 JMS 规范的“应用程序服务器工具”部分。
