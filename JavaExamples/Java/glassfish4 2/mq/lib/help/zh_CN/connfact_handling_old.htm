连接工厂对象属性 >
3.0 连接处理

使用“连接处理”选项卡，可以重新配置有关将由此工厂创建的连接的基本信息。

连接类型 使用下拉列表选择连接类型。

代理主机名 指定宿主客户机将连接的代理的计算机名称。

代理主机端口 指定端口映射器的端口号。客户机使用端口映射器提供的信息连接所需服务（jms 或 ssljms）。

代理服务端口 指定用来建立到 jms 或 ssljms 服务的直接连接的端口号。使用“代理服务端口”，可以绕过端口映射器。这样，客户机就可以使用单个已知端口通过防火墙进行连接。

HTTP URL 指定将用于连接消息服务的 URL。默认情况下，它是 http://localhost/imq/tunnel。
