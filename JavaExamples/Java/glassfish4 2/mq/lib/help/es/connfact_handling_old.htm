Gestión de conexiones 3.0

Use la ficha "Gestión de conexiones" para volver a configurar la información básica sobre las conexiones que se crearán mediante esta fábrica.

Tipo de conexión Use la lista desplegable para seleccionar el tipo de conexión.

Nombre de host del agente Especifique el nombre del equipo que aloja el agente al cual se conectará el cliente.

Puerto de host del agente Especifique el número de puerto del asignador de puertos. El cliente usa la información suministrada por el asignador de puertos para conectarse al servicio que desee, ya sea jms o ssljms.

Puerto de servicio del agente Especifica el número de puerto que se debe usar para efectuar una conexión directa con el servicio jms o ssljms. Use este puerto para evitar el uso del asignador de puertos. De esta manera, los clientes pueden conectarse a través de los cortafuegos usando un único puerto conocido.

URL HTTP Especifique el URL que se utilizará para conectarse al Servicio de mensajes. De forma predeterminada es http://localhost/imq/tunnel.
