Aggiungi oggetto ConnectionFactory > Override intestazione messaggio

Utilizzare questa scheda per ignorare i campi di intestazione messaggio JMS che specificano persistenza, durata e priorità dei messaggi. Poiché ignorare questi valori potrebbe interferire con il lavoro di utenti correnti o progettisti di applicazioni, coordinare il proprio lavoro con questi prima di eseguire le modifiche.

Ignora JMSDeliveryMode Specifica se il campo JMSDeliveryMode impostato dal client può essere ignorato. Predefinito: false.

Valore JMSDeliveryMode Specifica il valore di override di JMSDeliveryMode. I valori sono 1 (non persistente) o 2 (persistente). Predefinito: 2.

Ignora JMSExpiration Specifica se il campo JMSExpiration impostato dal client può essere ignorato. Predefinito: false.

Valore JMSExpiration Specifica il valore di override di JMSExpiration (in millisecondi). Predefinito: 0 (non scade).

Ignora JMSPriority Specifica se il campo JMSPriority impostato dal client può essere ignorato. Predefinito: false.

Valore JMSPriority Specifica il valore di override di JMSPriority (un numero intero da 0 a 9). Predefinito: 4 (normale).

Ignora messaggi sulle destinazioni temporanee Specifica se gli override si applicano a destinazioni temporanee. Predefinito: false.
