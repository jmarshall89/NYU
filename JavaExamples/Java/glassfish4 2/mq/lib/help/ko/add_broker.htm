브로커 추가

이 대화 상자를 사용하여 관리 콘솔의 탐색 창에 브로커의 표현을 만들 수 있습니다. 이를 위해 브로커를 시작할 필요는 없습니다. 브로커에 연결하려면 브로커를 시작해야 합니다.

브로커 레이블 관리 콘솔의 탐색 창에 표시할 이 브로커의 이름을 지정합니다. 이 이름이 브로커의 인스턴스 이름과 같을 필요는 없습니다.

호스트 이 브로커를 호스트하는 시스템의 이름을 지정하거나 기본값 localhost를 그대로 둡니다.

기본 포트 포트 매퍼에 지정되는 포트의 번호를 지정합니다. 기본값은 7676입니다. 호스트 이름과 기본 포트 번호를 조합하여 브로커를 고유하게 식별합니다. 같은 서버에서 브로커 인스턴스를 두 개 실행하려면 각 브로커의 포트 매퍼에 각각 다른 포트 번호가 있어야 합니다.

사용자 이름 사용자 이름을 지정합니다. 관리자의 경우 기본값은 admin입니다. 이 정보는 브로커에 연결을 시도할 때 인증에 사용됩니다. 이 대화 상자에서 암호를 입력하는 것은 안전하지 않습니다. 보안이 염려되는 경우 브로커에 연결하여 해당 정보를 입력하라는 메시지가 표시될 때까지 기다려야 합니다.

비밀번호 사용자 비밀번호를 지정합니다. 관리자의 경우 기본값은 admin입니다. 이 정보는 브로커에 연결을 시도할 때 인증에 사용됩니다. 이 대화 상자에서 암호를 입력하는 것은 안전하지 않습니다. 보안이 염려되는 경우 브로커에 연결하여 해당 정보를 입력하라는 메시지가 표시될 때까지 기다려야 합니다.

  • 입력한 모든 정보를 완결하고 브로커를 추가하려면 확인 버튼을 누릅니다.

추가한 브로커에 연결하려면 다음 작업을 수행합니다.

  1. 관리 콘솔의 탐색 창에서 브로커를 선택합니다.
  2. 작업 > 브로커에 연결을 선택합니다.

실행 중인 브로커를 쿼리 또는 업데이트하려면 다음 작업을 수행합니다.

  1. 관리 콘솔의 탐색 창에서 브로커를 선택합니다.
  2. 아직 연결되어 있지 않은 경우에는 브로커에 연결합니다.
  3. 작업 > 브로커 쿼리/업데이트를 선택합니다.

이 대화 상자에서 지정한 특정 브로커에 대한 정보를 변경하려면 다음 작업을 수행합니다.

  1. 탐색 창에서 브로커를 선택합니다.
  2. 브로커 연결이 이미 끊긴 상태가 아니면 연결을 해제합니다. 탐색 창에 브로커 아이콘 모양이 이면 연결이 끊긴 것입니다.
    브로커 연결을 해제하려면 작업 > 브로커에서 연결 해제를 선택합니다.
  3. 브로커 등록 정보 대화 상자를 열려면 작업 > 등록 정보를 선택합니다.
