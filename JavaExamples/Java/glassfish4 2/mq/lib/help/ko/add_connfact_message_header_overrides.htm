연결 팩토리 객체 추가 > 메시지 헤더 무시

이 탭을 사용하여 메시지의 지속성, 수명, 우선 순위를 지정하는 JMS 메시지 헤더 필드를 무시합니다. 이 값을 무시하면 현재 사용자 또는 응용 프로그램 설계자에게 방해가 될 수 있기 때문에 이 값을 변경하기 전에 먼저 상의하십시오.

JMSDeliveryMode 무시 클라이언트에서 설정한 JMSDeliveryMode 필드를 무시할 수 있는지 여부를 지정합니다. 기본값은 false입니다.

JMSDeliveryMode 값 JMSDeliveryMode를 무시하는 값을 지정합니다. 값은 1(비지속) 또는 2(지속)입니다. 기본값은 2입니다.

JMSExpiration 무시 클라이언트에서 설정한 JMSExpiration 필드를 무시할 수 있는지 여부를 지정합니다. 기본값은 false입니다.

JMSExpiration 값 JMSExpiration를 무시하는 값(밀리초)을 지정합니다. 기본값은 0(만료되지 않음)입니다.

JMSPriority 무시 클라이언트에서 설정한 JMSPriority 필드를 무시할 수 있는지 여부를 지정합니다. 기본값은 false입니다.

JMSPriority 값 JMSPriority를 무시하는 값(0에서 9까지의 정수)을 지정합니다. 기본값은 4(보통)입니다.

임시 대상에 대한 메시지 무시 임시 대상에 무시를 적용할 것인지 여부를 지정합니다. 기본값은 false입니다.
